# ❤️ Telegram Dating Bot

Telegram-бот знакомств на **aiogram 3** + **SQLite**.

Проект поддерживает:
- регистрацию анкеты с фото, геолокацией и описанием;
- поиск анкет по возрасту, полу и радиусу;
- лайки и суперлайки;
- взаимные симпатии;
- жалобы и блокировки;
- премиум-статус;
- оплату через **Telegram Stars**;
- админ-панель;
- реферальную систему;
- антиспам и валидацию данных.

---

## Возможности

### Для пользователей
- создание профиля через `/start`;
- редактирование анкеты;
- просмотр анкет других пользователей;
- лайки и суперлайки;
- просмотр взаимных симпатий;
- управление фильтрами поиска;
- покупка премиума и пакетов суперлайков;
- жалобы и блокировки.

### Для администратора
- список пользователей;
- статистика бота;
- рассылка сообщений;
- выдача премиума;
- бан / разбан пользователей.

Установка
1. Клонировать проект
Bash

git clone https://github.com/your-username/telegram-dating-bot.git
cd telegram-dating-bot
2. Создать виртуальное окружение
Bash

python -m venv venv
3. Активировать окружение
Windows
Bash

venv\Scripts\activate
Linux / macOS
Bash

source venv/bin/activate
4. Установить зависимости
Bash

pip install -r requirements.txt
5. Создать файл .env
Скопируйте содержимое из .env.example в .env и заполните значения.

6. Запустить бота
Bash

python main.py
Настройка .env
Минимально нужно указать:

BOT_TOKEN — токен Telegram-бота;
ADMIN_ID — Telegram ID администратора.
Если вы используете только Telegram Stars,
PAYMENT_PROVIDER_TOKEN можно оставить пустым.

Как получить BOT_TOKEN
Откройте @BotFather
Создайте нового бота
Скопируйте токен в .env
Как узнать ADMIN_ID
Например, через:

@userinfobot
или любым другим ботом, который показывает Telegram ID
Принцип работы
Регистрация
Пользователь проходит анкетирование:

имя;
возраст;
пол;
кого ищет;
геолокация;
фото;
описание.
Поиск
Алгоритм учитывает:

возраст;
пол;
радиус поиска;
блокировки;
уже просмотренные анкеты;
наличие фото.
Лайки
обычный лайк;
суперлайк;
взаимная симпатия создаётся автоматически при совпадении лайков.
Премиум
Премиум даёт:

5 суперлайков в день;
дополнительные возможности в будущем.
Платежи
Проект использует Telegram Stars.

Доступные покупки:
премиум на 30 дней;
пакет из 10 суперлайков.
Для Stars provider_token не нужен.

База данных
Бот использует SQLite и создаёт файл базы автоматически:

text

dating.db
Также создаются таблицы:

users
likes
views
blocks
reports
referrals
stats
payments
Безопасность
В проекте реализованы:

проверка имени и описания;
фильтрация ссылок;
антиспам;
проверка фото;
ограничение частоты действий;
защита от повторных лайков;
защита от лайка самому себе;
блокировки пользователей.
Геолокация
Для определения города по координатам используется
OpenStreetMap Nominatim.

Для работы этого функционала нужен доступ в интернет.

Примечания
Проект рассчитан на небольшой или средний бот.
Для больших нагрузок лучше вынести кеши и rate limit во внешнее хранилище.
SQLite хорошо подходит для старта, демо и небольшого production.
